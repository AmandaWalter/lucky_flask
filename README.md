# Lucky Flask

Installation:

0. These instructions should be executed in the shell.
1. Ensure python is installed on your system
2. Clone this repo or download [this zip](#)
3. Install `pip` via `easy_install` (python 2)
4. (Optional) If you want to keep your python packages separate (a good ideaâ€¦), then install [virtualenvwrapper](https://virtualenvwrapper.readthedocs.org/en/latest/) and `mkvirtualenv lucky_flask`.
5. Install dependencies via `pip install -r requirements.txt`
